什么是 SubHub？

SubHub 是一个免费的代理节点聚合与订阅格式转换平台。我们自动从多个公开的 GitHub 节点仓库采集节点，经过去重和协议解析后提供给用户使用。目前聚合了超过 4000 个节点，覆盖美国、日本、香港、新加坡、韩国、台湾、德国、英国、法国等 16 个国家和地区，支持 SS、VMess、Trojan、VLESS、Hysteria2 等主流协议。

支持哪些客户端？

SubHub 支持一键生成以下客户端的订阅链接：Clash（包括 Clash for Windows、Clash Verge、ClashX）、Mihomo（Meta 内核）、V2RayN、V2RayNG、Sing-Box、Surge、Shadowrocket、Quantumult X。你只需要选择客户端类型，复制订阅链接，导入即可使用。

订阅转换功能

除了提供免费节点，SubHub 还提供在线订阅格式转换服务。你可以将任意订阅链接或节点分享链接转换为 Clash YAML、V2Ray Base64、Sing-Box JSON、Surge 配置等格式。所有转换在服务端内存中完成，不存储任何用户数据。

完全免费开源

SubHub 基于 Cloudflare Pages + Workers + KV 构建，完全 Serverless 架构，零服务器成本。项目完全开源，你可以在 GitHub 上查看源代码，也可以一键 Fork 部署到自己的 Cloudflare 账号。
